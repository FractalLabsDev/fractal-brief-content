# Hello from Fractal Brief

Welcome to **Fractal Brief** - a simple way to share beautifully formatted documents.

## What is this?

Fractal Brief is a markdown document viewer created by [Fractal Labs](https://fractallabs.dev). It provides a clean, branded way to share documents without relying on external services like GitHub Gists.

## Features

- **Clean Design** - Minimalist interface that puts your content first
- **Multiple Themes** - Switch between nature, dark, and light modes
- **GitHub Markdown** - Full support for GFM including tables and code blocks
- **Fast Loading** - Content served from GitHub's CDN
